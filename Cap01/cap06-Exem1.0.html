Percorrer os caracteres de uma String

Podemos usar o .length para obter o comprimento de uma String.

Por exemplo, se declaramos uma variável chamada let cidade = "Pelotas", teremos: cidade.length = 7

Usando o chartAt, podemos acessar um elemento da string utilizando o seu indice. Por exemplo, cidade.chartAt(0) = P
